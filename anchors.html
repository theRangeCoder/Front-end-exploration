Anchors and attributes

Before we commence this topic, if you are starting off your journey with HTML, I would encourage you to visit this link to brush up some basic syntax and file structure concepts in HTML. The anchor tags are used to create hyperlinks in an HTML document. Attributes (such as href) are used to embed the internal or external link that the hyperlink should redirect to. Attributes are placed in the opening anchor tags. Any external link should start with http:// or https://.

Different types of paths:

1. Absolute paths: These refer to linking your website to external websites/ images.

2. Relative paths: These refer to linking pages/ images within your own website, such as linking the homepage to the 'About us' page of the same website. This can simply be done by putting the file name (with the path, if the file is in a subfolder) along with the attribute keyword.
