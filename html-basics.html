The strong tag

The strong tag is used to put strong importance on a part of a text. For example: This part of the text has a strong emphasis on it.

The emphasis tag

The emphasis tag is used to lay stress on a certain part of a text. For example: This part of the text has an emphasis on it.

Some interesting points on file naming and organization

The homepage should always be named index.html.

The main folder of the website is known as the root folder. The index.html file should always be in the root folder but the other files can be in subfolders as well, depending on how one wishes to stay organized.

Images should preferably be placed in a subfolder, to stay better organized.
